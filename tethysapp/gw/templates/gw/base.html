    Instructions

    Select Aquifer

    • Click the "Select Aquifer" dropdown and select the desired aquifer (or All aquifers)

    Specify Required Data

    • Move the slider below the heading "Minimum Number of Data Points per Well"
    • This slider adjusts how many wells will be dislayed. Only wells with the minimum specified number of measurements will be mapped.

    Display Wells

    • Click the "Display Wells" button. The app will load and then display the wells for the specified aquifer with the required measurements.

    Click on individual wells

    • When clicked, each well will display timeseries data and well metadata.
    • Use the "Clear Wells" button to clear all displayed wells.

    Display Surface

    • Click the "Display Surface" button to display the interpolated groundwater levels for the specified aquifer.
    • Use the slider at the bottom of the map to view the interpolated surface at different timesteps.
    • Use the "Clear Surface" button to clear all displayed groundwater surface maps.
